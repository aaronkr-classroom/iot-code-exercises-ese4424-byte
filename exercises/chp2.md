# chp2 연습문제


2. 마이크로파이썬의 펌웨어를 올리기 위해서 부팅할 때 별도의 버튼을 클릭하게 합니다. 해당 절차의 필요성을 설명하세요.
: 부팅 시 별도의 벼튼을 누르게 하는 이유는 일반 실행 모드와 펌웨어 업데이트 모드를 명확히 구분하기 위해서입니다. 보드는 평소에 사용자 코드를 실행해야 합니다. 만약 버튼 없이 자동으로 펌웨어 업데이트 모드로 들어가게 하면, 실수로 펌웨어가 덮어써지거나 장치가 정상 동작하지 않을 위험이 발생할 수 있습니다. 따라서 버튼의 역할은 버튼을 누르지 않으면 기존 프로그램이 실행되고, 버튼을 누르면 펌웨어 업데이트 모드로 진입하여 펌웨어를 업로드할 수 있도록 하여 의도적으로만 펌웨어를 업로드할 수 있도록 하는 안전장치와 같습니다. 부팅 시 버튼을 누르는 절차는 단순히 번거로운 것이 아니라 의도적으로 펌웨어 업로드를 보장하기 위해서 입니다.
